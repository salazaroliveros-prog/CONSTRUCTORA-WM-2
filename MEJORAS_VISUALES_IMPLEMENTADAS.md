## 🚀 **Cómo Usar los Efectos**

### **Aplicar Efectos Rápidos:**
```html
<!-- Tarjeta con múltiples efectos -->
<div class="interactive-card shimmer-effect particle-effect">
  Contenido
</div>

<!-- Botón con efecto líquido -->
<button class="btn-primary-enhanced btn-liquid">
  Acción
</button>

<!-- Icono con gradiente y glow -->
<div class="icon-box icon-gradient-blue">
  <Icon />
</div>
```

### **Badges Especiales:**
```html
<span class="badge-holographic">Holográfico</span>
<span class="badge-success">Éxito</span>
<span class="badge-warning">Advertencia</span>
```

### **Barras de Progreso:**
```html
<div class="progress-neon h-4">
  <div class="progress-neon-fill" style="width: 75%"></div>
</div>
```

### **Indicadores de Estado:**
```html
<div class="status-indicator status-online"></div>
<div class="status-indicator status-busy"></div>
```
